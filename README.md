## Installation

### Clone/Fork the Repo

- Create a fork of this repo and then clone this repo to your local machine using `git clone https://github.com/YOURUSERNAME/material-magic.git`

### Setup

> install dependencies

```bash
$ npm install
```

> your ready to go!

```bash
$ ng serve  or  ng serve --open
```
